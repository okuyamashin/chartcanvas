# 技術要件

## 使用技術
- 言語: JavaScript
- フレームワーク/ライブラリ: **依存ライブラリなし（ゼロ依存）**
  - ChartCanvasは外部ライブラリに依存しない純粋なJavaScriptライブラリです
  - DOMなしモード（headless mode）をサポートし、サーバーサイドやCLI環境でも動作します
  - Node.jsの標準機能（`vm`モジュール）のみを使用してCLIを実装しています
